# Политика конфиденциальности "Мои подписки"

Дата последнего обновления: 25 января 2026 года

## Общие положения

"Мои подписки" — это iOS-приложение для управления подписками и расходами. Мы стараемся максимально прозрачно объяснять, какие данные обрабатываются и как именно они используются.

Ниже описано простым и понятным языком, какие данные **не собираются**, какие данные **обрабатываются**, где они хранятся и кем обрабатываются платежи.

## Какие данные мы **НЕ собираем**

Приложение **не запрашивает и не обрабатывает**:

- имя, фамилию, отчество
- номер телефона
- адрес проживания
- данные документов
- контакты из адресной книги
- фотографии, файлы или медиаконтент
- данные банковских карт

В приложении:
- отсутствует регистрация
- отсутствует логин
- не требуется создание пользовательского аккаунта

## Какие данные обрабатываются

Для корректной работы приложения могут обрабатываться следующие данные:

- информация о подписках (название сервиса, стоимость, периодичность, даты списаний)
- настройки уведомлений
- технические параметры, необходимые для работы приложения

Эти данные **не используются для идентификации личности пользователя**.

## Хранение данных

Данные о подписках и настройках:

- хранятся на серверах приложения
- используются исключительно для обеспечения работы функциональности
- не передаются третьим лицам
- не используются в рекламных целях

Доступ к данным ограничен и защищён техническими мерами безопасности.

## Аналитика и технические данные

Для улучшения стабильности и качества приложения может использоваться обезличенная аналитика, например:

- Firebase Analytics

Такие сервисы могут автоматически собирать техническую информацию, включая:
- тип устройства
- версию iOS
- версию приложения
- данные о сбоях и ошибках

Эти данные:
- не содержат персональной информации
- не позволяют установить личность пользователя
- используются только для анализа и улучшения приложения

## Платежи и покупки

Все покупки и подписки в приложении осуществляются через **Apple In-App Purchases (StoreKit)**.

Это означает:
- данные банковских карт не передаются и не хранятся нами
- обработка платежей полностью осуществляется Apple
- мы получаем только информацию о статусе подписки (активна / неактивна)

## Реклама и передача данных

- Приложение **не использует рекламу**
- Мы **не продаём и не передаём данные** пользователей третьим лицам
- Данные не применяются для кросс-трекинга между приложениями или сайтами

## Удаление данных

Пользователь может удалить все связанные с приложением данные:

- удалив приложение с устройства
- либо обратившись к нам по контактам ниже с запросом на удаление данных

После удаления данные больше не используются.

## Изменения политики

Мы можем обновлять эту политику конфиденциальности. Актуальная версия всегда публикуется с новой датой обновления.

# Privacy Policy of MySubscriptions

Last updated: January 25, 2026

## General Information

MySubscriptions is an iOS application for managing subscriptions and expenses. We aim to be fully transparent about what data is processed and how it is used.

Below, we clearly explain which data is **not collected**, which data **may be processed**, where it is stored, and how payments are handled.

## Data We **Do Not Collect**

The application **does not request or process**:

- first name, last name, or other personal identifiers
- phone number
- home or mailing address
- government-issued identification documents
- contacts from the address book
- photos, files, or media content
- credit or debit card information

In the app:
- there is no registration
- there is no login
- no user account is required

## Data That May Be Processed

To ensure proper operation of the app, the following data may be processed:

- subscription information (service name, price, billing period, charge dates)
- notification settings
- technical parameters required for the app to function

This data **is not used to identify the user personally**.

## Data Storage

Subscription and settings data:

- are stored on the application’s servers
- are used solely to provide app functionality
- are not shared with third parties
- are not used for advertising purposes

Access to the data is restricted and protected by technical security measures.

## Analytics and Technical Data

To improve app stability and quality, anonymized analytics may be used, such as:

- Firebase Analytics

These services may automatically collect technical information, including:
- device type
- iOS version
- app version
- crash and error data

This data:
- does not contain personal information
- cannot be used to identify the user
- is used only to analyze and improve the app

## Payments and Purchases

All purchases and subscriptions in the app are processed via **Apple In-App Purchases (StoreKit)**.

This means:
- we do not receive or store credit card data
- payment processing is fully handled by Apple
- we only receive subscription status information (active / inactive)

## Advertising and Data Sharing

- The app **does not display ads**
- We **do not sell or share user data** with third parties
- Data is not used for cross-app or cross-site tracking

## Data Deletion

The user may delete all data associated with the app:

- by deleting the app from the device
- or by contacting us using the details below to request data deletion

After deletion, the data is no longer used.

## Changes to This Policy

We may update this Privacy Policy from time to time. The current version will always be published with an updated revision date.

## Contact Information

If you have any questions regarding privacy or data processing, you may contact us:

**Email:** puryaev.pavel@gmail.com
